# GitHub Push Steps

Repository name required by Future Interns:

```text
FUTURE_ML_02
```

Run these commands from inside the `FUTURE_ML_02` folder:

```bash
git init
git add .
git commit -m "Complete Future Interns ML Task 2 support ticket classification"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/FUTURE_ML_02.git
git push -u origin main
```

Before pushing, create a public GitHub repository named `FUTURE_ML_02`.

After pushing:

1. Copy the GitHub repository link.
2. Add it to `LINKEDIN_POST.md`.
3. Add it to the official Future Interns task submission portal when available.
